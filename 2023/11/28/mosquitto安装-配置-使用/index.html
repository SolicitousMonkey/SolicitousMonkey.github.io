2023-11-28|

改配置文件mosquitto.conf

创建/添加 用户

  1. 创建：mosquitto_passwd -c pwfile.example admin
  2. 添加：mosquitto_passwd pwfile.example admin

常用的几个指令参数

-c 加载指定的配置文件，即mosquitto.conf

-d 将服务放在后台执行

-p 监听指定端口的代理，并且不建议与-c选项一起使用

-v 启用详细日志模式
